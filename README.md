# oracle-multi

多人版占星週運展示（十二宮位簡化版）。

這個 repo 建議用來放「簡化、多使用者共用」的占星週運頁面，例如：

- 只提供十二星座的一般化週運，不需要輸入個人出生資訊。
- 或者作為個人版 `oracle-weekly-forecast` 的補充說明頁、導覽首頁。

實際前端可以沿用個人版的 `index.html` 結構，視需要調整成：

- 以星座選單或按鈕切換不同星座的十二宮位星等與摘要。
- 精簡掉個人出生時間／地址欄位，只保留說明文字與示意版十二宮運勢。

## 開發與部署

本 repo 同樣可以只包含一個 `index.html`，直接部署在 GitHub Pages 或任何靜態空間。
